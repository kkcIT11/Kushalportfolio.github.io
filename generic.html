How to Create a New User in Azure

Prerequisites

  • Global Administrator or User Administrator role in Azure AD.
  • Access to the Azure portal.
  • An active Azure subscription.

Steps to Create a User in Azure Entra ID

  1. Sign in to Azure Portal: Open your browser and go to Azure Portal and sign in using your administrator credentials.
  2. Navigate to Azure Active Directory: In the Azure portal, click on Azure Active Directory from the left-hand sidebar. If you cannot see it, use the search bar at the top to search for it.
  3. Create a New User: Under Azure Active Directory, click Users, then click the + New user button.
  4. Fill in User Information:
    • Identity: Select between Create user (for a new user) or Invite user (for external users).
    • Name: Enter the full name of the user.
    • Role: Select the user role (typically User by default).
  5. Set Password: Choose to auto-generate or manually set the user's password. Ensure the password complies with Azureâ€™s security policy. Optionally, check User must change password at next sign-in.
  6. Assign Licenses: Under Assigned licenses, click + Assign and choose any relevant licenses, such as Office 365.
  7. Assign Groups (Optional): Add the user to specific groups if required.
  8. Review and Create: Once all information is reviewed, click Create to finalize user creation.

Managing Users

  • Reset password, update profile information, and assign or revoke roles as needed.
  • Enable or disable user access to specific resources.

Troubleshooting

  • User Creation Failed: Ensure that the username is unique.
  • Password Issues: Verify the password policy and reset if necessary.
